# 📌 Project Overview – VAPT Week 2

## 🔐 Introduction
This project demonstrates a complete Vulnerability Assessment and Penetration Testing (VAPT) cycle performed on a Metasploitable2 virtual machine within a controlled lab environment.

The purpose of this assessment is to identify security weaknesses, validate them through exploitation, and provide actionable remediation strategies.

---

## 🎯 Objectives
- Discover open ports and running services
- Identify vulnerabilities in the system
- Perform exploitation to validate risks
- Gain system-level access (if possible)
- Document findings with proper evidence

---

## 🧪 Scope
- Target: Metasploitable2 (192.168.56.101)
- Environment: Local Virtual Lab
- Tools: Kali Linux-based toolset

---

## ⚙️ Tools & Technologies
- Nmap (Network Scanning)
- Nikto (Web Vulnerability Scanner)
- Metasploit Framework (Exploitation)
- Netcat (Manual exploitation)

---

## 🔍 Approach
The assessment follows industry-standard methodology:

1. Reconnaissance  
2. Scanning  
3. Exploitation  
4. Post-Exploitation  
5. Reporting  

---

## 🚨 Key Outcome
- Multiple vulnerable services identified  
- Critical misconfigurations detected  
- Root access successfully achieved via bind shell  
- System fully compromised in lab setup  

---

## 📊 Deliverables
- Detailed VAPT Report (PDF)
- Scan Results (Nmap, Nikto)
- Exploitation Evidence (Screenshots)
- Workflow Documentation

---

## ⚠️ Disclaimer
This project is conducted strictly in a controlled lab environment for educational and ethical purposes only.
